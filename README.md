## Installation
- Gehe zu: `C:\Users\DeinName\Documents\My Games\SubwaySim 2`
- Erstelle einen ordner mit dem Namen: `Mods`
- Entpacke dort die ZIP Datei. (Dessen inhalt dort hinein ziehen)
- - Nutze dafür  `WinRAR` oder `7Zip`
